About the Public Giveaway Bot

The Public Giveaway Bot by NDM Network is a powerful and user-friendly tool built to run seamless Discord giveaways. With smart automation, real-time updates, and rerolling, it's the perfect assistant for community engagement and promotional events.



🔹 Core Features

  • 🎁 Slash command-based giveaway creation
  • ⏰ Real-time countdown timers with live status
  • 🔁 Manual or auto rerolling
  • 🔒 Entry requirements (roles, server join date, etc.)
  • 📬 Claim buttons and winner DM notifications
  • 📝 Persistent giveaways via database support

🔹 Why It’s Public

We believe in open, accessible tools. That’s why the bot is free to use and maintained with transparency. You can join our Discord to suggest features, report bugs, or contribute to development.

🔹 Built With

Developed using Discord.js and hosted on Railway, the bot is designed for stability and speed.
